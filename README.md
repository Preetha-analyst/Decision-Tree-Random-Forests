# Task 5: Decision Trees and Random Forests

## Objective
Learn tree-based models for classification and compare Decision Tree and Random Forest algorithms.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-Learn

## Dataset
Heart Disease Dataset

## Procedure
1. Loaded and preprocessed the Heart Disease dataset.
2. Split the data into training and testing sets.
3. Trained a Decision Tree Classifier and visualized the tree structure.
4. Trained a Random Forest Classifier.
5. Compared the performance of both models using accuracy.
6. Analyzed feature importance.
7. Evaluated the model using cross-validation.

## Results

- Decision Tree Accuracy: 98.54%
- Random Forest Accuracy: 98.54%
- Cross Validation Score: 99.71%

## Conclusion

Both Decision Tree and Random Forest models achieved excellent performance on the Heart Disease dataset with an accuracy of 98.54%. The cross-validation score of 99.71% indicates that the model generalizes well and provides reliable predictions.

## Learning Outcomes

- Understood Decision Tree classification.
- Learned Random Forest ensemble learning.
- Compared model performance using accuracy.
- Explored feature importance analysis.
- Evaluated model robustness using cross-validation.
